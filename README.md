## 💡 使用指南 (How to Use)

每个 Group 文件通常包含以下部分：

1.  **话题详情 (Topic Details)**
    *   列出该组涵盖的所有具体题目及 "You should say" 提问要点。
2.  **核心串题思路 (Core Linking Logic)**
    *   解释如何用一个核心故事（Core Story）串联该组内的多个话题。
    *   分析不同话题之间的逻辑联系。
3.  **通用素材库 (General Material Library)**
    *   提供适配该组话题的英语范例（English Answer）。
    *   提供对应的中文翻译，辅助理解。
4.  **补充素材 (Supplementary Material)**
    *   部分文件包含“中英逐句对照版”，标注了断句节奏（`/` 短停顿，`//` 长停顿）和重读词汇，适合跟读练习。

## 🚀 备考建议

1.  **先看思路**: 阅读每个 Group 的“核心串题思路”，理解故事线。
2.  **个性化修改**: 参考“通用素材库”，将故事中的人名、地点、细节修改为自己的真实经历，增加真实感。
3.  **开口练习**: 利用“中英逐句对照版”进行跟读，注意语音语调和停顿。
4.  **模拟考试**: 随机抽取 Topic 进行 2 分钟独白模拟，检查是否能灵活运用素材。
